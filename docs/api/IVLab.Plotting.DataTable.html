    DataTable(String)

    Attempts to initialize a data table from the given csv filename, and initializes a random data table instead on failure.

    Declaration
    public DataTable(string filename)
    Parameters
    Type Name Description
    String filename

    Name of the csv file, excluding .csv.
    Remarks

    The csv file should be of the following form: Namely, its first column should be made up of data point names/IDs, its first row should be made up of column names, and the rest should be the actual numeric data values that will make up the table.
